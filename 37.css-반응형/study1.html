반응형 css

화면의 너비에 따라 다른 스타일의 css가 적용 되도록 스타일을 입력하는 방법

보통 media query라고 부른다

화면너비의 분기점(break point)

  • pc화면 보통 화면너비 최소값 1024px
  • tablet화면 - 768px 이상 1024px 미만
  • mobile 화면 - 768px 미만

모바일 css화면을 스타일링 할 때는 가로 사이즌느 되도록이면 상대단위 (%)를 사용

세로사이즈는 되도록이면 auto를 활용 -> 세로는 스크롤이 긴 화면은 좋지 않기 때문에 정확한 사이즈로 지정해 놓는 것은 비효율 적이다.
