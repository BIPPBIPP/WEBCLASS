함수 - function

자바스크립트에서 동작을 표현하는 데이터

function이라는 키워드를 통해 자바스크립트의 동작을 표현하는 함수를 만들 수 있다.

function 함수명(재료){동작할 소스코드}

  1. 함수는 정의를 먼저 해야 함.
  2. 정의한 함수를 실행함

함수의 매개변수 설정

함수는 매개변수(재료)를 입력받을 수 있는 형태로 만들 수 있다.

함수를 실행하는 실행부에서 함수명(인자) 형태로 함수를 실행할 수 있다.

매개변수의 개수는 제한이 없다.

함수 안에서 return

함수 안에서 return 구문을 쓰게되면 return 구문 오른쪽의 결과를 함수가 사용됬던 그 자리에 남기고 함수는 그 즉시 종료된다.

리턴값은 함수 내부에서 함수 외부로 데이터를 전달할때 사용하는 값이다.
